## Superheroes
Welcome to the Flask Code Challenge - Superheroes! In this challenge, you will build an API for tracking heroes and their superpowers using Flask. Below is a summary of the project setup, features, requirements, installation, and usage.

### Features
Add, view, edit, and delete heroes in a contact book.
Establish relationships between heroes and superpowers.
Validations for hero powers and descriptions.
Route endpoints for retrieving hero and power data.
### Requirements
Before running the Flask Code Challenge, make sure you have the following:

Python 3.x installed on your system. You can download Python from python.org.
### Technologies used: 
Python, Sqlite3,flask

### Installation
To get started, clone this repository to your local machine:


``
git clone https://github.com/winny52/Superheroes.git
Change to the project's directory:

``
### Usage

To use the Flask Code Challenge - Superheroes:

Make sure you are in the project directory.

Run the Flask API:
python app.py

Use your preferred method (Postman, browser, or provided React frontend) to interact with the API.
